see

An alternative to Python’s dir function. Easy to type; easy to read! For humans only.

Install

To download and install the latest release, simply run:

# easy_install -U see

Alternatively, to install from source:

# python setup.py install

If you don’t have root privileges, append --home="$HOME" to install to your home directory.

Usage

>>> from see import see
>>>
>>> see()
  os   see()   sys
>>> see([1, 2, 3])
  []   in   +   +=   *   *=   <   <=   ==   !=   >   >=   hash()   help()
  iter()   len()   repr()   reversed()   str()   .append()   .count()
  .extend()   .index()   .insert()   .pop()   .remove()   .reverse()   .sort()
>>> see('a string', '.is*')
  .isalnum()   .isalpha()   .isdigit()   .islower()   .isspace()   .istitle()
  .isupper()
